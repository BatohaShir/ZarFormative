# Настройка SSH-ключа для GitHub (Windows)

## Шаг 1: Создание SSH-ключа

1. Открой терминал в VS Code (нажми Ctrl+`)
2. Введи команду (замени email на свой, который используешь на GitHub):

   ssh-keygen -t ed25519 -C "твой@email.com"

3. Когда спросит куда сохранить — просто нажми Enter (сохранит в стандартную папку)
4. Когда спросит пароль (passphrase) — можно просто нажать Enter дважды (без пароля)

---

## Шаг 2: Запуск SSH-агента

Введи эти команды по очереди:

   eval $(ssh-agent -s)
   ssh-add ~/.ssh/id_ed25519

---

## Шаг 3: Копирование ключа

Введи команду чтобы показать ключ:

   cat ~/.ssh/id_ed25519.pub

Выдели ВЕСЬ текст который появился (начинается с "ssh-ed25519") и скопируй его (Ctrl+C)

---

## Шаг 4: Добавление ключа в GitHub

1. Открой в браузере: https://github.com/settings/keys
2. Нажми зелёную кнопку "New SSH key"
3. В поле "Title" напиши название (например: "Мой компьютер")
4. В поле "Key" вставь скопированный ключ (Ctrl+V)
5. Нажми "Add SSH key"
6. Подтверди паролем от GitHub если попросит

---

## Шаг 5: Проверка подключения

Введи в терминале:

   ssh -T git@github.com

Если спросит "Are you sure you want to continue connecting?" — напиши "yes" и нажми Enter

Должно появиться: "Hi ИмяПользователя! You've successfully authenticated..."

---
